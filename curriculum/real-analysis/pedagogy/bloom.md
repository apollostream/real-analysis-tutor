# Bloom's Taxonomy Applied to Real Analysis

Source of truth: `docs/source-guides/RealAnalysis_CompleteGuide_V2.1.md`,
Part III C, "Bloom's Taxonomy Applied to Real Analysis."

Bloom's Taxonomy provides a framework for calibrating question depth.
Real Analysis mastery requires systematic progression through all six
levels for each major result.

## Critical failure mode

The critical failure mode of most AI tutors — and most students studying
alone — is defaulting to the bottom two levels: Remember and Understand.

## The six levels

| Bloom Level | Example (Uniform Continuity) | Example (Compactness) | Target Phase |
|---|---|---|---|
| Remember | State the definition with all quantifiers explicit. | State the open cover definition of compactness. | Every phase |
| Understand | Explain why delta must not depend on x. What breaks if it does? | Explain what "every open cover has a finite subcover" is asserting. | Every phase |
| Apply | Prove f(x) = sqrt(x) is uniformly continuous on [1, ∞). | Use Heine-Borel to determine whether (0,1] is compact. | Rungs 1–2 |
| Analyze | Which hypothesis in Heine-Cantor is essential? Trace where compactness is invoked. | Where exactly does EVT invoke compactness? | Rungs 3–4 |
| Evaluate | Is your proof the minimum correct argument? Where is it doing unnecessary work? | Is there a shorter proof of Heine-Borel? | Rung 4 |
| Create | Construct a function continuous on R but not uniformly continuous. | Construct a set bounded but not compact in an unusual metric. | Rung 5 |
